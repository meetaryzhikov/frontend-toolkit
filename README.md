# My Frontend Toolkit

A small collection of pragmatic frontend utilities, starters and guides focused on scalable UI architecture and team-friendly practices.

## What’s inside
- `packages/component-library` — UI primitives + Storybook
- `packages/hooks` — useful React hooks
- `packages/lint-config` — shareable ESLint + Prettier configs
- `examples` — demo apps using the toolkit
- `docs` — guides, checklist and migration notes

## Quickstart
1. Clone this repo
2. `cd packages/component-library && npm install && npm run dev`
3. Open Storybook at `http://localhost:6006`
